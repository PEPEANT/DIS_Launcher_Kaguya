운영 원칙

현재 운영 기준은 게임 점수와 후쥬 지급을 분리하고, 일상 운영을 이벤트·공지 중심으로 처리하는 것입니다.

게임 점수는 각 게임 랭킹용이고, 후쥬는 이벤트·운영 지급용입니다.

일상 운영은 로비 공지, 이벤트 메시지, 운영 지급을 우선 사용하고 레거시 정산 센터는 기존 데이터 보정이나 예외 처리에만 사용하세요.
